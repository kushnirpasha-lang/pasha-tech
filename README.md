# Pasha Tech — сайт-визитка

Сайт-визитка для сервиса «цифровое присутствие для русскоязычных мастеров в Израиле».

**Live:** https://kushnirpasha-lang.github.io/pasha-tech/

## Стек

- Чистый HTML + CSS + JavaScript (без фреймворков)
- Шрифты: Inter + Heebo (Google Fonts)
- Деплой: GitHub Pages

## Локальный запуск

Открой `index.html` в браузере — никакой сборки не нужно.

## Структура

```
index.html      — вся разметка (10 блоков)
styles.css      — стили, тёмная тема, адаптив
script.js       — FAQ-аккордеон, анимации, демо-чат
assets/         — изображения и иконки
```

## Деплой

GitHub Pages публикует автоматически из ветки `main`.
